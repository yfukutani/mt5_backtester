# バックテスト検証ログ

## EA概要: RSI_Reversal

逆張り系EA。以下の3シグナルをOR条件で組み合わせ、MA200トレンドフィルターで方向を絞る。

| シグナル | 買い条件 | 売り条件 |
|---|---|---|
| RSI逆張り | RSI≤27.5に達した後、≥30に回復 | RSI≥75に達した後、≤72.5に低下 |
| BB逆張り | 終値がBB-2.5σ以下に達した後、バンド内に回復 | 終値がBB+2.5σ以上に達した後、バンド内に回復 |
| ダブルパターン | ダブルボトム検出→ネックライン上抜け | ダブルトップ検出→ネックライン下抜け |

**共通フィルター:** MA200（上昇トレンド時のみ買い、下降トレンド時のみ売り）  
**固定設定:** SL=50pips / TP=100pips（2:1）/ LotSize=0.01

### 構造的課題の診断

全ての損益管理改善が失敗した根本原因は、**エントリーシグナルの勝率の低さ（25-35%）**にある。

- 勝率25-35%でTP/SL = 2.0〜2.4倍のRR比は、理論的には期待値がプラスになりうる
- しかし実際には「勝てる年・相場」と「負ける年・相場」の差が極めて大きい（2024年 +30,742円 vs 2020年 -18,701円）
- 損益管理を変更すると勝率は改善するが、利益幅（RR比）が下がり、好成績年の利益を削ってしまう
- フィルターは「悪い取引を除去」する前に「良い取引も除去」してしまう

### どのような状況で有用か

この戦略（RSI逆張り + BB2.5σ + ダブルパターン + MA200）が機能する相場環境：

| 条件 | 具体例 | 理由 |
|---|---|---|
| **明確なレンジ相場** | USDJPY 140-160円台での往来（2023-2024年） | 過熱したRSI・BB外れが信頼できる反転シグナルになる |
| **高ボラティリティ** | ATR（H4）25pips以上 | TPに届くだけの値動きが確保できる |
| **MA200が横ばい〜緩傾斜** | トレンドが穏やか | MA200フィルターがよく機能し、誤方向エントリーを排除できる |
| **FRB・日銀の政策が安定期** | 利上げサイクルが完成した後など | 急激な一方向トレンドが出にくく、逆張りの有効性が上がる |

逆に機能しない環境：

| 条件 | 具体例 | 理由 |
|---|---|---|
| **強トレンド相場** | 2020-2021年のコロナ後USDJPYの方向感のない低ボラ期 | RSI過熱域に届かず、届いても反転せずそのまま延伸 |
| **一方向の長期トレンド** | 2022年の円安急進（110→150円） | MA200が追いつかず、逆方向シグナルが頻発 |
| **低ボラティリティ** | ATR（H4）20pips未満 | TPに届く前にSLに掴まる、値動きが小さくシグナルが不正確 |

### 運用推奨

1. **運用開始前に直近3ヶ月の年度別パフォーマンスを確認**（2024年のような好環境かチェック）
2. **定期再検証（3〜6ヶ月ごと）を必須とする**（過去2〜3年のデータでバックテストを再実施）
3. **損益管理の改変は行わない**（ブレークイーブン・トレーリング・ATR-SL全て悪化実績あり）
4. **現在の固定SL/TPが最良**（SL=50/45pips、TP=110/105pips、RR比≈2.2〜2.3）
5. **EURUSDは採用を慎重に検討**（2016-2022年は一貫してマイナスが多い）
