# バックテスト検証ログ

## EA概要: RSI_Reversal

逆張り系EA。以下の3シグナルをOR条件で組み合わせ、MA200トレンドフィルターで方向を絞る。

| シグナル | 買い条件 | 売り条件 |
|---|---|---|
| RSI逆張り | RSI≤27.5に達した後、≥30に回復 | RSI≥75に達した後、≤72.5に低下 |
| BB逆張り | 終値がBB-2.5σ以下に達した後、バンド内に回復 | 終値がBB+2.5σ以上に達した後、バンド内に回復 |
| ダブルパターン | ダブルボトム検出→ネックライン上抜け | ダブルトップ検出→ネックライン下抜け |

**共通フィルター:** MA200（上昇トレンド時のみ買い、下降トレンド時のみ売り）  
**固定設定:** SL=50pips / TP=100pips（2:1）/ LotSize=0.01

## 却下した戦略・ペアとその理由

| 対象 | 理由 |
|---|---|
| GOLD（XAUUSD） | 2024-2025の強い上昇トレンドでMA200フィルターが機能せず（買いシグナルのみ、RSIが過売りに届かない） |
| GBPUSD | どの設定でもSharp0.36以下、DD大きい |
| GBPJPY | H1で-10,711円、H4で+934円（PF1.03）と不安定 |
| EURJPY | H1・H4ともマイナス |
| BB ANDフィルター（v2.0試作） | RSI極端ゾーンとBB±2σ超えが高相関のため追加効果なし |
| ダブルパターン（EURUSD/EURJPY/GBP系） | MA200フィルターあり・なし両方で大幅マイナス |
