## 5. 確立した主要結論

- **月利フロンティア:** 必要DD% = 0.72 ÷（年間リターン/DD比）。現状ブックでは月利6%はレバ非依存で**最大DD55-68%**を伴う。現実的射程は**月利2-3%（年率30-45%）**。
- **増レバはリターン/DDを上げない:** net と DD を同率で動かすだけ。綺麗にスケールするのは PB USDJPY のみ。
- **GOLDは PB GOLD が最適:** 別戦略7案すべて棄却。深い調整＋高コントラクトに短期・タイトストップ順張りが正解。
- **新規無相関源の探索は収穫逓減:** 規律下（OOS/IS両プラス必須）で生存した新枠は VBO/ETH の2つのみ。
- **運用ルール:** 新戦略は **OOS/IS 両期間プラスを必須ゲート**とする。
