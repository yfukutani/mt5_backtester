## 第1弾の学びと第2弾の設計方針

**機能した唯一の軸 = 時間とレンジの設計**（金曜/NY午後/0時台/MinRange正規化）。
**全滅した軸 = 介入系**（エグジット改良・エントリー確認・ダマシ回避・停止規律・ML）。
**未検証の宝 = 逆発見4件**——不整合ブレイクPF1.86 / 弱いバーPF2.14 / 同時シグナルPF4.44 /
DD直後PF3.27。これらは「削るフィルター」としては使えないが、**「厚くするサイズ調整」としては
一度も検証していない**。第2弾はここを本丸に、時間深掘り・レンジ定義・市場拡張・方法論保証を固める。

**評価原則（継続）:** every_tick / 1施策1変更 / IS・OOS両立ゲート / プラトー確認 /
ログ事前分析→EA実装の2段構え。サイズ調整系は**過学習リスクが高いため通常より厳格に**
（IS/OOS/銘柄横断の3重一貫を要求、メタラベリング失敗の教訓）。

## S案第1バッチ 検証結果（2026.07.05）— ★主目標達成: F2-3採用

### ✅ 採用: F2-3 リバーサル型増しロット（3銘柄・本番config反映済み）

レンジ窓内ドリフトと逆方向のブレイクでロット2倍（`UseReversalBoost`/`Boost_Mult=2.0`）。

| 銘柄 | Boost無し | **Boost有り（採用）** | 変化 |
|---|---|---|---|
| GOLD | +203,317/PF1.703/DD11.1% | **+239,109/PF1.725/DD13.9%** | +17.6% |
| USDJPY | +8,589/PF1.05 | +12,286/PF1.064 | +43% |
| GBPJPY | +15,327/PF1.04 | **+39,576/PF1.086** | **+158%** |
| 合算 | +227,233 | **+290,971** | **+28.0%** |

- 検証プロセス: ログ複合スコアリング→3重一貫ゲート（IS+33,398/OOS+27,968/3銘柄全プラス）→
  DD効率評価（ポートフォリオ効率1.38）→EA実装→every_tick実測（+28.0%＝ログ合成予測+27.0%と一致）
- 論理: アジア時間のポジション調整と逆向きのブレイクは、欧州本流の初動である確率が高い
  （第1弾C1逆発見の順活用。リバーサル型はOOSでもPF1.41と期間安定）
- 運用ノート: 増しロットでDDも増える（GOLD 11.1→13.9%）。DD予算固定運用では
  GlobalLotMult側で全体を約0.9倍に調整すれば同DD・純益+15%相当
