# BTC第4バックログ — 約1000格子（応答曲面方式・400案分の不採用理由から設計）

**作成日:** 2026.07.12
**設計転換:** 過去3ラウンド400案で確定した事実——(1)再現性のあるエッジは**funding逆張り（G1）ただ一つ**、
(2)採用に至ったのは全て**実証済み構造の改善**、(3)個別アイデアの列挙は生存率0.5-2%。
よって第4弾は「アイデア1000個」ではなく、**実証済みメカニズムの応答曲面を約1000格子で事前登録し
全格子を機械的に評価する**方式に切り替える。これは(a)採用済みG1の頑健性の地図を作り、
(b)同メカニズムの未踏領域（ETH×正常化退出・多銘柄・Bitfinexマージン）を系統的に踏破するため。

## 不採用理由 → 格子設計への反映

| 教訓（ラウンド） | 反映 |
|---|---|
| 点最適は罠・プラトーのみ採用（全R） | **採用単位は「格子の点」でなく「近傍領域」**: 主要2軸の3×3近傍が全て[両+/J両+]の領域のみ生存 |
| %とJPYの逆転（R2 V7） | 全格子でJPY固定ロット換算を必須列に |
| 適応閾値は改善しない（R3 M1/Q7） | 閾値軸は固定値の帯для地図作成（適応形は含めない） |
| 週次n不足（R2 COT） | 全格子が日次データ由来 |
| ETH版G1は固定5日退出で死亡（R2 P5） | **FEファミリーの主仮説: v1.2型退出（funding正常化）がETH版を救うか** |
| 単発t≥2でも多重比較で偽陽性（R1 D4等） | 格子全体の陽性率を帰無分布（シャッフル）と比較・領域要件・WF二段 |
| 市場中立/オーバーレイ/OHLCV凍結（全R） | 凍結領域の格子はゼロ |

## ファミリーと格子の事前登録（合計 約1,010格子）

### FG. BTC funding応答曲面（384格子）— 採用済みG1の頑健性地図
- 入口形（12）: 固定閾値 {-0.003, -0.004, -0.005, -0.006, -0.007} / 分位 {q3, q5, q7} /
  持続 {2日連続<-0.004, 3日連続<-0.004} / ギャップ {fd-fd30<q5} / Bybit q5
- 退出形（8）: 固定 {3, 5, 7, 10日} / funding>0 / >med90 / >med180 / 正常化+固定7上限
- サイズ（2）: 1x / 深度2x（<-0.006で倍）
- 執行（2）: 翌日寄付（現行） / 翌日陽線確認後
- **目的:** 採用形（-0.004×med90×1x×寄付）が「広い正の台地の中」にあることの確認。
  ⚠️ **G1本体の変更は提案しない**（地図作成が目的。変更候補が現れてもWF+テスター+ジャッジの
  三重ゲートを別途通す）

### FE. ETH funding応答曲面（384格子）— 主仮説「正常化退出がETH版を救う」
- FGと同一の格子構造（閾値・分位はETH自身の分布で再計算）
- R2でETH版G1（固定5日退出）は実費PF0.8で死亡。しかしBTCではv1.2退出が+57%改善した——
  同じ改善がETHに効くなら新スリーブ候補になる

### FX. 多銘柄funding移植（72格子）— メカニズムの横展開
- 銘柄（6）: LTC / XRP / SOL / BNB / DOGE / ADA（XMで取引可能なものを優先。BNBはXM非提供=
  スクリーニングのみで参考記録）
- 入口（3）: 各銘柄自身の {q3, q5, q7}
- 退出（4）: 固定5日 / funding>0 / >med90 / 固定3日
- 判定は銘柄毎のJPYゲート（低価格銘柄はminlotのノーショナルで再計算）
- ⚠️ 過去教訓「アルト横展開は全滅（trend-hold）」——ただしG1はメカニズムが違う（清算圧力は
  銘柄固有のfundingで直接観測される）ため1回だけ系統的に確認する価値がある

### BK. マルチソース合成（16格子）

### BF. Bitfinexマージン（40格子）— R3から追補継続

### EX. 執行最適化（32格子）

## 評価プロトコル（全格子共通・事前登録）

1. **一次評価（Python・全格子）:** イベント非重複化→n / %平均 / t / 前半後半（2021.06分割）/
   **JPY固定ロット合計（前後半とも）** / 対照超過
2. **領域生存:** 主要2軸（入口×退出）の3×3近傍が全て「両半+かつJ両+」→領域として生存。
   孤立セルは自動棄却（多重比較対策）
3. **帰無比較:** シグナル日をランダム置換した帰無格子の陽性率と比較し、実陽性率が帰無の
   2倍未満ならファミリーごと棄却
4. **二次評価（生存領域のみ）:** ウォークフォワード（閾値等を前年までのデータで決定）で
   フォールド過半プラス
5. **三次評価:** 代表セル（領域の中心・最良でない）をテスター実費（スワップ込み）で確認
6. **採用提案:** 既存G1との相関・イベント重複率を実測し、ユーザージャッジへ
