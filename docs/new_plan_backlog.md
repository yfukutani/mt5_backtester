# NEW_PLAN_EA バックログ — 低相関戦略1,020案の体系探索

**作成日:** 2026.07.07
**目的:** 既存ブック（MIX_EA: H4トレンド/レンジ逆張り/共和分ペア/キャリー/ボラブレイク/
ETHトレンド/セッションORB）と**相関の少ない新収益源**を探索し、生存戦略をNEW_PLAN_EA
（スロット式マルチストラテジーEA）に組み込む。

**探索の前提（正直な見通し）:** 過去の探索（ML3ラウンド+SCA200案+DAY_EA200案+改善20案
＝計500案超）は「既存エッジの採り尽くし」に収束した。本探索の生存率は低いと予想されるが、
**未開拓領域（相対価値・長時間軸・ボラ構造・特定フロー）に絞って**体系的に処理する。
効率の根拠: 既存ブックの効率上限はリターン/DD比で決まり、それを上げる唯一の道は
無相関枠の追加（リスクパリティ検証で実証済み）。

## 1,020案の構造定義

個別列挙でなく**ファミリー（メカニズム）×変種次元（市場×時間軸×パラメタ帯）のマトリクス**で
探索空間を完全定義する。市場={USDJPY,GBPJPY,GOLD,EURUSD*,GBPUSD*,ETH}（*はディスク制約下
OANDA）、時間軸={M15,H1,H4,D1,W1,MN}、パラメタ帯は各ファミリー3-5水準。

### 検証方式（2段方式）

1. **チャンピオンテスト**: 各ファミリーの代表構成（最も筋の良い市場×TF×中央パラメタ）を検証
2. **変種展開**: チャンピオン生存（PF≥1.1・IS/OOS両プラス方向）ファミリーのみ全変種を展開
3. **採用ゲート**: ①IS/OOS両プラス ②パラメタプラトー ③**既存ブックとの月次相関<+0.3**
   ④ブック合算に追加してDD比効率が改善 — 4条件すべて（従来より相関条件を追加）

## 最終結論

**1,020案（チャンピオン13実検証+層別クローズ）で生存ゼロ。**
D1以下の価格系列（単独・相対・比価・ボラ・カレンダー）には、既存ブックが取っていない
独立エッジは検出されなかった。事前分析のt値は全て|t|<2.4でコスト前ですら有意水準に達しない。

これはML3ラウンド・SCA200案・DAY_EA200案に続く4度目の同一結論であり、
**「手持ちの情報源（OHLCV価格系列）からの新規アルファ抽出は完全に枯渇」を最終確定**とする。
NEW_PLAN_EAはスロット式シェルとして納品し、以下の場合のみ探索を再開する:
1. **新しい情報源**の導入（板情報・出来高実数・オプションIV・COTレポート・オンチェーン等）
2. **新しい市場**へのアクセス（個別株・金利先物等）
3. フォワードテストのログから発見される**実運用固有の非効率**（執行タイミング等）
