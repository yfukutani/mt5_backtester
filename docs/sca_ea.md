# SCA_EA — セッションORBスキャルパー（GOLD/USDJPY/GBPJPY・M15）

**作成日:** 2026.07.03
**位置づけ:** 既存ブック（フォワード移行済み）とは**全く別の新規EA**。資金30万円・月利5%目標で開発。
**メカニズム:** アジア時間（サーバー0-9時）のレンジを定義し、ロンドン序盤のブレイクを日中完結で取る
オープニングレンジ・ブレイクアウト（ORB）。収益源はセッション構造のボラ拡大＝既存ブックのどの枠とも別軸。

## 検証原則

**スキャルピングは全検証を`every_tick`（ヒストリカルスプレッド実費込み）で実施**（open_pricesはSL/TPの
バー内約定を正しく再現できないため不可）。全期間2016-2026・M15・XMフィード。1回約1分で反復可能。

### 中核的発見：エッジは「広レンジ日」に集中する

リスク%サイジング（全取引を等リスク化）でPFが1.11→0.72に反転したことから、**利益は広レンジ日の
ブレイクに集中し、狭レンジ日のブレイクは有毒**と判明。固定ロットはレンジ幅に比例したP&Lで自然に
広レンジ日を大きく張るため機能していた。この構造を順方向に活かしたのが`MinRange_ATRd`引き上げ
（狭レンジ日の除外）で、全銘柄の質を一段改善した。**リスク%サイジングはこのエッジと構造的に非互換＝
固定ロット厳守。**

### 月利5%目標への正直な回答

**月利5%は本エッジでは生存可能なDDで到達不能。** 5%/月＝10年で+1.8M（対30万）に必要なサイズでは
最大DDが元本の約95%超（破綻圏）に達する。17実験で到達した実測フロンティアは：

- **月利1.2%（DD22%）〜2.0%（DD37%）が実用域**、外挿で4.0%（DD75%・非推奨）
- スキャルピングはスプレッド実費（every_tickで全額課金済み）がエッジの大半を食う構造で、
  GOLD ORBのPF1.44は実はこの制約下で優秀な部類

**推奨:** ミックスB（GOLD0.03/JPY各0.01・月利1.19%期待・DD22%）でフォワード開始。
本EAの位置づけは「単独で月5%を出す道具」ではなく「既存ブックと無相関の追加収益源」が適切
（セッション構造ドリブン＝H4/D1ブックと収益タイミングが独立）。

## デプロイ

- チャート: 各銘柄のM15に`SCA_EA`をアタッチ（GOLD/USDJPY/GBPJPY、Magic 20261002/20261000/20261001）
- config: `configs/sca_gold_m15.yaml` / `sca_usdjpy_m15.yaml` / `sca_gbpjpy_m15.yaml`
- サーバー時間依存（XM=GMT+2/+3前提のセッション時刻）。他ブローカーでは`RangeStartHour`等の
  時刻入力を要調整（OANDA展開時はサーバー時刻を確認のこと）
- `MaxSpreadPoints`はライブでは設定推奨（USDJPY 30、GBPJPY 50、GOLD 60目安）＝指標時の異常
  スプレッドでのエントリー防止
