# BTC戦略バックログ — 200案（BTC固有構造ベース・優先度付き）

**作成日:** 2026.07.12
**前提（正直な出発点）:** BTCは既に5系統（trend-hold / PB押し目 / VBOブレイク / SCAセッションORB /
trend-hold+ヒステリシス）を検証しすべて棄却済み。共通の死因は**「IS(2021-26)のみ機能・OOS(2015-21)
で負け」の時間不安定**——2017バブル→2018-84%暴落のサイクルがFX由来のテンプレートと構造不適合。
よって本バックログは「FX戦略の移植」ではなく**BTC固有の構造**（24時間365日市場・週末ギャップ無し・
半減期サイクル・ATH心理・機関化による性質変化・独自の外部情報源）を軸に設計する。

**「有効である」の定義:** 本リストは検証候補であり有効性の保証ではない。採用は従来ゲート
（every_tick or D1実費・**IS/OOS両プラス**・プラトー・既存ブック相関<+0.3）を通過したもののみ。

**環境制約:** XM BTCUSDで取得可能なのはOHLCV+tickvol+スプレッド（2015〜）。
funding rate・オンチェーン・取引所フロー等は**外部データ導入が前提**（G系=待機枠として分離）。
